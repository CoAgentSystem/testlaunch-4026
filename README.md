# testlaunch-4026

TestLaunch is a streamlined platform designed to help users quickly validate and prototype startup ideas by providing an intuitive interface for creating, testing, and sharing mockups with real-time feedback. It focuses on simplicity and speed, enabling entrepreneurs to iterate on concepts without technical barriers, fostering early-stage innovation and collaboration.

## Tech Stack
Next.js, Node.js, PostgreSQL, Tailwind CSS

## Setup
Run `docker-compose up` to start the infrastructure.
